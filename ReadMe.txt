1. UnZip the giving zip into any location.
2. launch the index.html using the Chrome browser.
3. Use the "Choose File" button on the page to select any sample text from the system. For convinience, there is a file
named "sample.txt" in the base folder of the project.
4. The output is listed in the Palindrome Reader Output section.
5. If you need to reload the file, "Reset File" button needs to be used. Reset File option will clear the output and clear the file reader.
